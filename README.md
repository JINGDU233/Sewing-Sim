# 缝纫机导针及紧线机构运动仿真

> 机械原理课程设计 —— 缝纫机导针机构与紧线机构的运动学分析与可视化仿真软件

## 项目简介

本软件基于 Python 开发，用于缝纫机导针及紧线机构的运动学分析与可视化仿真。通过对曲柄滑块机构和曲柄摇杆机构的运动学求解，实现对针杆和紧线头的位移、速度、加速度的精确计算，并提供实时动画和图表展示。

- **导针机构**：曲柄滑块机构（O₂AB），驱动针杆沿导轨上下往复运动
- **紧线机构**：曲柄摇杆机构（O₂CDO₃），带动紧线头 E 点来回摆动
- **传动系统**：外啮合齿轮传动，主动轮 O₁ 驱动从动轮 O₂

程序主界面分为两部分：
- **左侧**：参数输入面板，支持 10 组预设方案、自定义参数、参数校验

## 功能特性

- **10 组预设方案**：内置十组备选设计参数，一键加载对比
- **实时动画仿真**：机构按实际运动规律实时动画展示，各杆件颜色区分，铰接点与滑块清晰可见
- **E 点轨迹绘制**：随动画逐帧绘制紧线头运动轨迹
- **运动学分析图表**：自动计算并绘制 E 点轨迹、针杆位移、速度、加速度随曲柄转角变化曲线
- **实时状态显示**：底部状态栏实时显示当前曲柄转角、针杆位置、摇杆角度
- **速度调节**：支持 0.1x ~ 5.0x 无极调速
- **运动学数据导出**：支持导出完整周期运动学数据为 CSV 文件
- **输入校验**：实时校验参数格式、取值范围，机构装配条件（Grashof 条件等）验证

## 运动学模型

### 曲柄滑块机构（导针）

已知曲柄 O₂A = H/2，连杆 AB = O₂A / (O₂A/AB)。给定曲柄转角 φ，求解滑块 B 点在 Y 轴上的位置：

```
B_y = A_y - √(AB² - A_x²)
```

### 曲柄摇杆机构（紧线）

已知 O₂O₃、O₃D，通过极位共线法解算出曲柄 O₂C 和连杆 CD。已知曲柄转角 (φ+β₁)，求解 D 点（两圆交点）和 E 点（∠CDE = β₂）：

```
D = 圆(O₃, O₃D) ∩ 圆(C, CD)
E = D + DE · rotate(CD方向, β₂)
```

速度和加速度通过中心差分法数值求解。

## 校验规则

- **基本校验**：参数非空、正数、角度 0~360° 范围
- **装配校验**：
  - 导针机构：AB > O₂A
  - 齿轮中心距：O₁O₂ ≥ L
  - 摇杆角度：α'' > α'
  - Grashof 条件：最短杆 + 最长杆 ≤ 其余两杆之和，且最短杆为曲柄
  - 齿轮中心距取整偏差 < 0.5mm
